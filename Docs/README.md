

# Docs

This directory contains development and operational documentation for the Exrela System.

## Purpose

The documents in this directory support the design, implementation, maintenance, and evolution of Exrela System.

They complement the formal specifications but do not replace them.

## Contents

- 00 — Workspace Architecture
- 02 — Source Management
- 90 — Development Guide
- 91 — Runtime Implementation Map
- 92 — Runtime Module Specification
- 93 — Runtime Development Log

## Canonical Specifications

The official system specifications are maintained in:

- T1_Definition
- T2_Runtime

## Notes

- Docs contains developer-facing guidance and implementation support.
- T1_Definition defines the system philosophy and specifications.
- T2_Runtime defines the Runtime Suite.
- Research, Archive, and Future materials are managed outside this directory.
