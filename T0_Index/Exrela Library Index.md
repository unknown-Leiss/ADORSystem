LibraryはInformation Source Architectureに従って構成される。

- Core Source
- Specification Source

---

## Library Principles

- Definitionはシステムの目的と思想を定義する。
- RuntimeはDefinitionを実現する方法を定義する。
- Researchは理論を研究・検証する。
- Archiveは履歴を保存する。
- Futureは将来構想を管理する。

Layer Boundaryを維持し、各カテゴリは他カテゴリの責務を代替してはならない。
